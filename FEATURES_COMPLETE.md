## 🚀 Technology Stack

### Backend
```
Node.js + Express.js
├── MongoDB (Mongoose)
├── JWT Authentication
├── bcryptjs (Password Hashing)
├── Cloudinary (File Storage)
├── Multer (File Upload)
└── CORS
```

### Frontend
```
React 18 + Vite
├── React Router DOM (Routing)
├── Tailwind CSS (Styling)
├── Axios (HTTP Client)
├── Recharts (Charts)
├── jsPDF (PDF Generation)
└── Context API (State)
```

### Infrastructure
```
Deployment: Vercel
Database: MongoDB Atlas
Storage: Cloudinary
```
